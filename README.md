# posiflora-assistant

Плагин протокола AI-ассистента для команды Posiflora.

## Что делает

- **SessionStart hook** — автоматически запускает стартовый ритуал в начале каждой сессии.
- **Skill `posiflora-protocol`** — подробный протокол работы с vault по запросу.

## Установка

```
/plugin marketplace add drozhzhinds/assistant-plugin
/plugin install posiflora-assistant@posiflora
```

## Обновления

Владелец пушит изменения в GitHub → пользователи запускают `/plugin marketplace update`.
